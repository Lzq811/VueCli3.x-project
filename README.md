# @VUECLI init project

project desc: @VUE/CLI + element-ui + echarts + less + 环境变量

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn start  #   默认 test 环境
yarn start--test
yarn start--prod
```

### Compiles and minifies for production
```
yarn build  #   默认 prod 环境
yarn build--test
yarn build--prod
```

### Lints and fixes files
```
yarn lint
```
